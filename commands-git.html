* Pour cloner un repo : - git clone "url du repo sur github" exemple ici : git clone https://github.com/ryhad-boughanmi/git-begins * Ensuite pour pouvoir push les modifs sur le repo git, on va : - git add "nom du fichier" ou bien -> 'git add .' (pour tout ajouter) - git commit -m "commentaire avec le commit" - git push -u origin main * Quand on a fait des changements a distance sur GitHub, on doit ensuite fair eun pull pour avoir les dernieres modifs: - git pull * Pour creer une nouvelle branche on fait : - git branch name_new_branch ici >> git branch add-create-branch-command * Puis pour changer de branche : - git checkout name_branch_we_want_to_go_to ici >> git checkout add-create-branch-command
